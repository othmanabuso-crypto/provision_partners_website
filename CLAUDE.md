# Provision Partners Website

## Project

Build a modern corporate website for Provision Partners Company Limited.

## Business

Provision Partners is a multi-sector supply company providing:

* Office Materials Supply
* Construction Materials Supply
* IT Equipment & Accessories Supply

## Website Goal

The website should:

* Build trust
* Showcase services
* Generate quote requests
* Present a premium professional image

## Design Style

* Modern
* Premium
* Corporate
* Clean
* Responsive

Avoid:

* Outdated designs
* Cluttered layouts
* Excessive animations

## Branding

Use the company logo as the primary branding reference.

Primary colors should be derived from the logo.

## Future Requirements

* GitHub repository
* cPanel deployment
* SEO optimization
* WhatsApp integration
* Quote request form
